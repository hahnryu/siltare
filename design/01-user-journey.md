# 01. 유저 여정

마지막 업데이트: 2026-03-04

---

## 등장인물

| 역할 | 설명 | 연령대 |
|------|------|--------|
| 요청자 | 링크를 만들어 보내는 사람. 자녀. | 30~50대 |
| 인터뷰이 | 이야기를 하는 사람. 부모님. | 60~80대 |
| 셀프 유저 | 자신의 이야기를 직접 기록하는 사람. | 30~60대 |

---

## 셀프 모드 여정 (나부터 시작)

```
siltare.app 방문
    ↓
"내 이야기 시작하기" 클릭
    ↓
[첫 방문] 온보딩 화면
  - 왜 하는가 (3개 카드)
  - 10시간 = 책 한 권 proposition
  - 챕터 그리드 + 세션 진도 시각화
  - 오늘 할 것 (1장 · 첫 번째 대화)
    ↓
[재방문] 바로 대화 화면
    ↓
AI 첫 메시지 (오리엔테이션 포함)
    ↓
10분 대화 (1챕터 1회차)
    ↓
"오늘은 여기까지" 클릭
    ↓
완료 화면
  - 진도 확인 (1장 1/3 완료)
  - 다음 세션 예고
  - 카카오 연결 유도 (재방문 알림용)
    ↓
[3~5회 반복 후 1챕터 완주]
    ↓
챕터 완주 화면
  - 1장 초고 생성
  - 자기발견 인사이트
  - 나만의 챕터 구성 제안 (2~9장)
  - "가족에게도 보내보세요" 유도
    ↓
[10챕터 완주]
    ↓
책 완성
```

## invite 모드 여정 (요청자)

```
siltare.app 방문
    ↓
"부모님께 보내기" 클릭
    ↓
/request 페이지
  [메인] 음성 입력: "누구한테 보내실 건가요?"
  [폴백] 4단계 텍스트 폼
    ↓
AI 파싱 결과 확인 카드
    ↓
인사 녹음 (선택)
    ↓
링크 생성
    ↓
카카오톡 전송 / 링크 복사
    ↓
[알림] 부모님이 이야기를 시작하셨습니다
    ↓
/archive/{id} 에서 이야기 확인
    ↓
분석 리포트 / 책 주문
```

## invite 모드 여정 (인터뷰이 · 부모님)

```
카카오톡 링크 수신
    ↓
/i/{id} 동의 화면
  [첫 방문] "이야기를 시작하겠습니다"
  [재방문] "지난 이야기를 이어하겠습니다"
    ↓
/interview/{id}
  [첫 방문] 자녀 인사 녹음 재생 → AI 첫 메시지
  [재방문] 이전 대화 표시 → 이어서
    ↓
10~30분 대화
    ↓
"오늘은 여기까지"
    ↓
완료 화면
  - 카카오 연결 유도
  - 링크 저장 안내
    ↓
/archive/{id} 결과 확인
```

## 인터뷰 상태 전이

```
pending
  → active (대화 시작)
    → session_end (오늘은 여기까지)
      → active (이어하기)
        → chapter_complete (챕터 완주)
          → active (다음 챕터 시작)
            → ... (반복)
              → complete (최종 완료)
```

---

## 재방문 동기 3가지

1. 완성 욕구: 챕터 진도 게이지, 책이 되어간다는 느낌
2. 자기발견: 챕터 완주 후 "나도 몰랐던 나" 인사이트
3. 결과물: 초고가 손에 잡히는 것

---

## 온보딩 플로우 상세

**첫 방문 (레이어 2 → 레이어 1)**
- 레이어 2: 온보딩 화면 (왜/어떻게/오늘)
- 레이어 1: AI 첫 메시지 (오리엔테이션 + 첫 질문)

**재방문 (레이어 1만)**
- 바로 AI 메시지 "지난번 이야기를 이어갑니다"
- 이전 대화 표시 + 날짜 구분선
