## 셀프 모드 여정 (나부터 시작)

```
siltare.app 방문
    ↓
"내 이야기 시작하기" 클릭
    ↓
[첫 방문] 온보딩 화면
  - 왜 하는가 (3개 카드)
  - 10시간 = 책 한 권 proposition
  - 챕터 그리드 + 세션 진도 시각화
  - 오늘 할 것 (1장 · 첫 번째 대화)
    ↓
[재방문] 바로 대화 화면
    ↓
AI 첫 메시지 (오리엔테이션 포함)
    ↓
10분 대화 (1챕터 1회차)
    ↓
"오늘은 여기까지" 클릭
    ↓
완료 화면
  - 진도 확인 (1장 1/3 완료)
  - 다음 세션 예고
  - 카카오 연결 유도 (재방문 알림용)
    ↓
[3~5회 반복 후 1챕터 완주]
    ↓
챕터 완주 화면
  - 1장 초고 생성
  - 자기발견 인사이트
  - 나만의 챕터 구성 제안 (2~9장)
  - "가족에게도 보내보세요" 유도
    ↓
[10챕터 완주]
    ↓
책 완성
```

## 인터뷰 상태 전이

```
pending
  → active (대화 시작)
    → session_end (오늘은 여기까지)
      → active (이어하기)
        → chapter_complete (챕터 완주)
          → active (다음 챕터 시작)
            → ... (반복)
              → complete (최종 완료)
```

## 재방문 동기 3가지

1. 완성 욕구: 챕터 진도 게이지, 책이 되어간다는 느낌
2. 자기발견: 챕터 완주 후 "나도 몰랐던 나" 인사이트
3. 결과물: 초고가 손에 잡히는 것

## 온보딩 플로우 상세

**첫 방문 (레이어 2 → 레이어 1)**
- 레이어 2: 온보딩 화면 (왜/어떻게/오늘)
- 레이어 1: AI 첫 메시지 (오리엔테이션 + 첫 질문)

**재방문 (레이어 1만)**
- 바로 AI 메시지 "지난번 이야기를 이어갑니다"
- 이전 대화 표시 + 날짜 구분선

## 이름 수집 플로우 (신규)

```
페이지 로드
  ↓
AI 첫 메시지 (프론트 직접 렌더링)
  "안녕하세요. 실타래입니다. 성함이 어떻게 되세요?"
  ↓
유저가 음성으로 이름 말함
  ↓
Whisper 전사 결과가 입력란에 자동 채워짐
  ↓
[전사 완료 직후 딱 한 번 힌트 표시]
  "잘못 받아적었다면 아래 키보드로 직접 고쳐주시면 돼요.
   앞으로 모든 과정 마찬가지예요."
  (localStorage로 한 번만 표시 제어)
  ↓
전송 시 이름 저장 (10자 제한)
  ↓
일반 대화 시작
```

**이름 수집 효과:**
- 마이크 사용법 자연스럽게 학습
- 음성 + 교정 조합 학습
- 이름 수집
- 첫 메시지 연결 오류 제거

## 1챕터 1세션 2단계 구조

**STEP 1. 생애 개관 (오프닝)**
- "태어나서 지금까지, 어떤 시절들을 지나오셨나요?
   길게 안 하셔도 돼요. 어떤 굵직한 시간들이 있었는지만."
- AI가 내부적으로 어느 시기에 말이 많아지는지 파악
- 깊이 파고들지 않음. 한 번만 가볍게 확인 후 STEP 2.

**STEP 2. space 레이어 진입**
- "그 중에서 가장 이른 기억부터 시작해볼게요."
- 여기서부터 레이어 구조대로 깊이 파고듦

## archive 페이지 초고 편집 기능 (신규)

**위치:** AI 요약 아래, 전체 대화 보기 위에

**구조:**
```
[섹션 제목] ✦ 1장 초고
[설명] AI가 대화를 바탕으로 작성한 초안입니다. 직접 다듬어 주세요.

[읽기 모드]
  초고 텍스트 표시
  [편집하기] 버튼

[편집 모드]
  <textarea> 초고 편집
  [저장] [취소] 버튼
```

**노출 조건:**
- `autobiography_draft[1]`이 있을 때만 섹션 표시
- 챕터 완주 전에는 섹션 자체가 없음

**저장 플로우:**
- 편집 후 저장 시 `/api/update-draft` 호출
- `{ interviewId, chapterNum: 1, draft: editedText }`

**포지셔닝:**
- "AI가 작성한 초안, 직접 다듬어 주세요"
- 완성품이 아닌 편집 가능한 출발점
