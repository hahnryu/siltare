# 05. 수익 모델

마지막 업데이트: 2026-03-04

---

## 핵심 원칙

수익은 분석에서 나온다. 대화는 항상 무료.

첫 대화는 진입 장벽 없이. 감정적으로 투자된 후 결제.

---

## 가격 구조

| 티어 | 가격 | 내용 |
|------|------|------|
| 무료 | ₩0 | 대화 1회, 텍스트 아카이브, 공유 링크 |
| 기록 보관 | ₩9,900 | 음성 원본 영구 보관, AI 요약, 챕터 자동 생성 |
| 성격 분석 | ₩19,000 | 1인 성격 프로파일, 가치관 맵, 인생 타임라인 |
| 관계 분석 | ₩29,000 | 2인 관계 다이나믹, 반복 패턴, 세대 비교 |
| 자서전 책 | ₩79,000~ | 소프트커버, 표지 커스터마이징, 배송 |
| 이어하기 | ₩9,900/회 | 2회차부터 유료 |
| 가족 구독 | 월 ₩9,900 | 무제한 대화, 분석, 음성 보관 |

## 전환 퍼널

```
무료 대화 완료 (100%)
  ↓
아카이브 확인 (70%)
  ↓
음성 원본 보관 구매 (15%)  ← ₩9,900 (가장 낮은 허들)
성격 분석 구매 (8%)        ← ₩19,000
  ↓
"엄마도 해보세요" 바이럴 (20%)
  ↓
관계 분석 구매 (5%)        ← ₩29,000
```

## 아카이브 페이지 CTA 순서

1. 음성 원본 보관하기 ₩9,900
2. 성격/가치관 분석 리포트 ₩19,000
3. 자서전 책으로 만들기 ₩79,000
4. 이어서 대화하기 ₩9,900
5. 가족에게 공유하기 (무료, 바이럴)
