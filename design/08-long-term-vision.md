## 한 줄 비전

합성 불가능한 인간의 육성을 수집하고, 연결하고, 인류의 집단 기억 지도를 만든다.

---

## 4단계 발전 경로

```
1단계: 개인의 목소리
  한 사람의 생애 인터뷰.
  음성 원본 + 전사본 + AI 요약.
  합성 불가능한 원본 데이터.
  (지금 여기)

2단계: 연결되는 서사 (온톨로지)
  인터뷰에서 추출된 인물, 장소, 시간, 사건이 자동 연결.
  A의 "부산 영도 피난"과 B의 "부산 영도 피난"이
  같은 사건이면 두 서사가 엮인다.
  → Human Voice Graph

3단계: 집단 기억의 코퍼스
  수만 개의 인터뷰가 쌓이면
  세대별, 지역별, 관계별 패턴이 드러난다.
  합성 불가능한 원본 데이터로 구축되는 집단 기억의 지도.

4단계: 읽기가 쓰기를 부른다
  타인의 이야기를 읽는다.
  같은 세대, 같은 동네, 같은 시대의 낯선 가족.
  공감하는 순간 우리 부모님에게도 실타래를 보내게 된다.
  아카이브가 플랫폼이 되는 지점.
```

---

## HOMP (Human-Origin Memory Protocol)

실타래는 HOMP의 첫 번째 애플리케이션이다.

```
HOMP (인프라)
  → 실타래 (제품)
    → 뿌리깊은나무 연구소 (연구)
```

세 레이어가 하나의 비전 아래 작동한다.

**온톨로지 구조**
모든 인터뷰에서 언급된 인물, 장소, 시간, 사건이 연결된다.
인터뷰가 쌓일수록 그래프가 풍부해지고,
데이터의 가치가 비선형으로 증가한다.

Human Voice Graph의 종착점:
개별 가족의 기록이 아니라 인류 공통 서사의 지도.

## 원본성의 해자

합성 데이터 시대에 "이 음성은 진짜 사람이 말한 것"이라는 증명이 가치를 갖는다.

**온체인 타임스탬프**
- 인터뷰 완료 시점에 해시값을 온체인에 기록
- "이 음성은 이 시점에 이 사람이 말한 것"을 증명
- 음성 복제가 일상이 되는 시대의 희소성 기반
- NodeONE Inc. 기존 인프라 위에서 구현

**왜 중요한가**
AI 합성 음성이 판별 불가능해지는 시점이 온다.
그 전에 원본성을 타임스탬프로 고정한 데이터는
대체 불가능한 자산이 된다.

## 글로벌 확장 전략

**왜 아리랑인가**
아리랑은 이별, 고향, 그리움의 서사.
한국인의 집단 무의식이자 보편 서사의 원형.
모든 문화권에 부모가 있고, 모든 부모는 죽고,
모든 자녀는 "그때 물어볼 걸"이라고 후회한다.
이 감정에 국경이 없다.

**확장 순서**
```
한국 (효도 선물)
  → 재외동포 730만명
    → 일본 (敬老の日, 경로의 날)
      → 중화권 (重陽節, 중양절)
        → 미주 디아스포라
          → 전 세계
```

AI 인터뷰는 언어만 바꾸면 전 세계에서 작동한다.

**BTS 아리랑 앨범 연계**
3.20 출시에 맞춰 #arirang 캠페인으로 글로벌 진입.

## 보편 서사 추출

수십만 개의 인터뷰가 쌓이면
문화와 언어가 달라도 반복되는 서사 구조가 보인다.

전쟁, 이주, 가난, 사랑, 후회, 감사.
세대를 관통하는 패턴.
그것이 인류의 얼굴이다.

**활용 방향**
- 큐레이션된 이야기 → 뉴스레터, 팟캐스트, 숏폼
- "부산 영도 피난" 검색 → 서로 다른 가족의 시선 교차
- 공공 서사의 정당성 기반 데이터
