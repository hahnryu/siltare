## 업데이트 원칙

새 기능을 추가하거나 구조가 바뀔 때마다:
1. 영향받는 모듈 파일만 업데이트
2. 마지막 업데이트 날짜 갱신
3. 07-decisions.md에 결정 이유 기록

## Claude Code 업데이트 프롬프트

기능 추가 후:
```
/design 폴더를 읽어줘.
오늘 추가한 기능: [기능 설명]
영향받는 모듈 파일만 업데이트해줘.
코드는 건드리지 말고 /design 파일만.
마지막 업데이트 날짜를 오늘로 바꿔줘.
```
